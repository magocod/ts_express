## Project setup

### Install dependencies

```bash
npm install
```

### Translate code to javascript (dist)

```bash
npm run build
```

### Run tests

```bash
# all
npm run test
npm run test:watch

# with coverage
npm run test:coverage
```

### Code style

```bash
# eslint
npm run lint

# code style
npm run format:check
npm run format
```
