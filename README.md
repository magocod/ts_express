## Project setup

### Install dependencies

```bash
npm install
```

### build

```bash
npm run build
```

### Run tests

- run all
```bash
npm run test
```

- run everything in parallel
```bash
npm run test:parallel
```

- coverage
```bash
npm run test:coverage
```

### Code style

- eslint check
```bash
npm run lint
```

- eslint fix
```bash
npm run lint:fix
```

- prettier check
```bash
npm run format:check
```

- prettier format
```bash
npm run format
```

npx typedoc
