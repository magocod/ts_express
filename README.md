## Project setup

```
npm install
```

### Translate code to javascript (dist)

```
npm run build
```

### Run www server

```
npm run start
```

### Run tests

```
// all
npm run test
npm run test:parallel

// with coverage
npm run test:coverage
```

### Run dev server

```
npm run dev
```

### Code style

```
// eslint
npm run lint

// code style
npm run format:check
npm run format
```
