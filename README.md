## Project setup

### Install dependencies

```bash
npm install
```

### Translate code to javascript (dist)

```bash
npm run build
```

### Run tests

```bash
npm run test
npm run test:parallel

npm run test:coverage
```

## Code style

### Eslint

```bash
npm run lint
```

### Prettier

```bash
npm run format:check
npm run format
```
